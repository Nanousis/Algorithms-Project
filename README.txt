Παναγιώτης Νανούσης, Γιάννης Γιαννούλας, Ορέστης Πανόπουλος.

Το πρόγραμμα αυτό είναι η απάντηση του project 2023 
για το μάθημα Αλγόριθμοι

Για να τρέξετε το πρόβλημα αρκεί να κάνετε make και
ύστερα ./main ακολουθούμενο σαν argument από το όνομα του test που θέλουμε να τρέξουμε
Για παράδειγμα ./main tests/test1

Για να κάνουμε εξατομικεύσεις στον κώδικα, 
μπορούμε να πάμε στο αρχείο printFunctions.c 
και να επιλεξούμε τι παραπάνω θέλουμε να εκτυπωθεί 

Το αρχείο main.c περιέχει τον κορμό του προγράμματος
Το αρχέιο mainFunctions.c περιέχει τις βασικές συναρτήσεις 
που χρησιμοποιεί η main.c

Το functions.h είναι το βασικό header file του προγράμματος 

Τα .py αρχεία είναι για να φτιαχτούν διαγράμματα σχετικά με τις τιμές μας
